Robot Memory
Benchmark Landscape

Most manipulation benchmarks are Markovian: the current frame contains everything needed to act. A new and fast-moving cluster of 2026 benchmarks deliberately breaks that assumption — the policy must remember what it saw, where things were, what it already did, or what happened in a previous trial. This atlas maps that emerging landscape: 6 memory-dependent benchmarks, the memory taxonomy they target, and which memory types each one probes. The field is young, so treat the specifics as preliminary.

Why a landscape, not a leaderboard. These benchmarks are weeks-to-months old and report few comparable model numbers yet — building per-model leaderboards would over-claim. Instead this atlas catalogs the benchmarks and organizes them by the kind of memory they stress, so the structure stands even as scores arrive. Memory mixes several axes; the canonical split by timescale is below, and the per-benchmark memory-type coverage follows.
