绝大多数操作基准都是马尔可夫的：当前帧已包含决策所需的全部信息。然而，2026 年涌现出一批快速发展的新基准，它们刻意打破了这一假设——策略必须记住它看到过什么、物体曾在何处、自己已经做了什么，或上一次试验中发生了什么。本图谱描绘了这一新兴格局：6 个依赖记忆的基准、它们所针对的记忆分类体系，以及每个基准各自考察哪些记忆类型。该领域尚处于起步阶段，因此请将其中的具体细节视为初步结论。

为何是全景，而非排行榜。这些基准均诞生于数周至数月之内，目前尚未报告足够多可供比较的模型数据——构建逐模型的排行榜会带来过度宣称。因此，本图谱转而对这些基准进行编目，并按其所考察的记忆种类加以组织，使得即便分数陆续到来，这一结构依然成立。记忆涉及若干维度；下文先给出按时间尺度划分的经典分类，随后是逐基准的记忆类型覆盖情况。

● = 基准明确针对该记忆类型
Temporal（时序）（事件顺序）与 Spatial（空间）（记住离开视野的位置）是覆盖最广的两个维度；Cross-trial（跨试验）记忆——在不同情节之间携带知识——则最为稀少且最具挑战，仅有 RoboMemArena 与 MemMimic 对其进行考察。
SOURCE · 基准清单与分类体系来自 Awesome-Memory-for-Robotics（benchmarks 部分）。基准：MIKASA-Robo-VLA (2502.10550), RoboMemArena (2605.10921), RoboMME (2603.04639), RMBench (2603.01229), LIBERO-Mem (2511.11478), MemMimic (2604.18933)。记忆类型标签依据各基准声明的侧重点进行分配；任务数量按其所报告的口径列出（LIBERO-Mem 与 MemMimic 并未以固定数量作为标志）。此层级尚属新兴——数据为初步结果，后续将有所变动。
