在本合集中，其他图谱各自覆盖一个任务套件，而 ManiSkill3 是一种引擎级聚合器：单一的 GPU 并行仿真器（构建于 SAPIEN 之上），用一套 API 统一了 12 个不同任务域 — 从桌面抓取到人形运动、灵巧手、软体，乃至真实到仿真的数字孪生 — 并覆盖 20+ 种机器人本体。通过在 GPU 上同时并行仿真与渲染，它可达 30,000+ FPS，相较以往平台提速 10–1000×、显存占用减少 2–3×。

这里的"聚合"指什么。 ManiSkill3 并非像 RoboVerse 那样迁移基准来聚合数据集 — 它把多种任务族聚合进同一引擎。同一套 GPU 并行内核、观测/奖励 API 与示范数据流水线服务于以下每一个任务域，因此无论本体是 Franka 机械臂、四足机器人 还是 人形机器人，策略都可以用完全相同的方式训练与评估。数百万帧示范数据由运动规划、强化学习与遥操作生成并随框架提供。
