Where the leaderboard atlases each cover one task suite, ManiSkill3 is an engine-level aggregator: a single GPU-parallelized simulator (built on SAPIEN) that unifies 12 distinct task domains — from table-top grasping to humanoid locomotion, dexterous hands, soft bodies, and real-to-sim digital twins — across 20+ robot embodiments behind one API. By parallelizing both simulation and rendering on the GPU it reaches 30,000+ FPS, running 10–1000× faster with 2–3× less GPU memory than prior platforms.

What "aggregator" means here. ManiSkill3 does not aggregate datasets the way RoboVerse migrates benchmarks — it aggregates task families into one engine. The same GPU-parallelized core, observation/reward API and demonstration pipeline serve every domain below, so a policy can be trained and evaluated identically whether the embodiment is a Franka arm, a quadruped, or a humanoid. Millions of demonstration frames ship from motion planning, RL, and teleoperation.
