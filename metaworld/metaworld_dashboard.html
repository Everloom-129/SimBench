Meta-World is the standard benchmark for multi-task and meta reinforcement learning: 50 distinct manipulation skills on a single Sawyer arm, each with randomized goals, sharing one table and one action space by design. The same 50 tasks are recombined into named evaluation configs — MT1/MT10/MT50 (one policy, many skills) and ML1/ML10/ML45 (learn to adapt to held-out skills). This atlas maps every task to the configs it belongs to and renders the canonical MT10 multi-task leaderboard.

One skill set, many benchmarks. Meta-World's design goal was structured diversity: 50 tasks varied enough to be distinct but sharing enough structure to enable transfer. MT configs test learning many skills with one policy; ML configs test generalization to unseen skills (train on N, adapt to 5 held-out). Each task also has 50 random goal positions. Success is binary per episode; the headline metric is mean success rate over a config's skills.
