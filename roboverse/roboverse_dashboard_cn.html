RoboVerse
聚合图谱

Jie Wang · everloom-129.github.io · GRASP Lab, UPenn  ·  与 Claude Code 联合创作  ·  2026.6.1

与其他条目作为单一基准不同，RoboVerse 是一个聚合平台：其核心 MetaSim 在一套与仿真器无关的配置系统之下抽象了多种不同的仿真器，进而将现有基准迁移为单一的统一格式。最终形成一个标准化的平台、数据集与基准，覆盖 6 个仿真器后端与 18 个来源基准 — 仅操作类迁移就产出了 276 个任务类别、约 50 万条轨迹以及 5.5k 个资产（用于策略学习的状态转移超过 5000 万次）。

此处"聚合"的含义。 RoboVerse 并不发明新任务 — 而是将它们统一起来。来自多种仿真器的资产、场景与轨迹被重构进 MetaSim 的通用接口，从而使同一任务可以跨后端实例化，同一策略可以在统一协议下评估。276 task categories / ~500k trajectories 这两项数据描述的是操作类迁移；导航与运动类来源通过同一平台进行整合。
