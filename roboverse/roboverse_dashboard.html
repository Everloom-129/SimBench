Where the other entries are individual benchmarks, RoboVerse is an aggregator: at its core, MetaSim abstracts diverse simulators behind one simulator-agnostic configuration system, then migrates existing benchmarks into a single unified format. The result is a standardized platform, dataset, and benchmark spanning 6 simulator backends and 18 source benchmarks — with the manipulation migration alone yielding 276 task categories, ~500k trajectories, and 5.5k assets (over 50M state transitions for policy learning).

What "aggregation" means here. RoboVerse does not invent new tasks — it unifies them. Assets, scenes, and trajectories from many simulators are restructured into MetaSim's universal interface, so the same task can be instantiated across backends and the same policy evaluated under one protocol. The 276 task categories / ~500k trajectories figures describe the manipulation migration; navigation and locomotion sources are integrated through the same platform.
